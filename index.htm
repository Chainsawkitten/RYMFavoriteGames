Scoring formula: Sort by: Show:

The sum of all votes. A vote is worth 51 - position. So the 1st position is worth 50, 2nd 49, and so on until position 50, which is worth 1 point.

The sum of all votes, multiplied by the number of votes. A vote is worth 51 - position. So the 1st position is worth 50, 2nd 49, and so on until position 50, which is worth 1 point.

The average of the game's votes. A vote is worth 51 - position. So the 1st position is worth 50, 2nd 49, and so on until position 50, which is worth 1 point.

A variant of instant-runoff voting where each user's first place vote is counted. Games with the lowest amount of votes are then discarded, and all votes for them are discarded (so if a user voted for it as their 1st game, their vote now becomes their 2nd game). The final score is how many turns it took to discard the game.
